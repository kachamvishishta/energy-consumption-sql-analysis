# Energy Consumption SQL Analysis

## Overview
This project analyzes energy consumption data using SQL to identify trends and insights.

## Tools Used
- SQL (MySQL)
- Aggregations
- Joins
- Subqueries

## Key Insights
- Year-wise energy consumption trends
- Region-wise comparison
- Highest and lowest consumption analysis

## Skills Demonstrated
- Data querying
- Data analysis
- Problem-solving using SQL
